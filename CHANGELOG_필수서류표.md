# 필수 서류 안내 표 추가

## 📋 추가 내용

확인서 하단에 **필수 서류 상세 안내** 표를 추가하여 작성자가 필수 서류의 법적 근거와 목적을 명확히 인지하도록 개선했습니다.

## 🎯 추가 목적

1. **법적 근거 명시**: 각 서류의 법적 근거를 명확히 제시
2. **목적 이해**: 왜 필요한지 구체적으로 설명
3. **인지도 향상**: 단순히 "미스캔"이 아니라 법적 의무임을 인식
4. **교육 효과**: 확인서 자체가 교육 자료 역할
